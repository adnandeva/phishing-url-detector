# Phishing URL Detection using Lexical Analysis

## Overview
This project detects whether a URL is legitimate or malicious using lexical features extracted directly from the URL string.

## How it Works
- URLs are loaded from a dataset  
- Labels are converted into binary (legit vs malicious)  
- Features are extracted from URLs:
  - URL length  
  - Number of dots  
  - Presence of HTTPS  
  - Number of digits  
  - Special characters
